# Run? Run. Run!

## Run?

Install the following packages using your system's package manager (we'll find
them at compile time using `pkg-config`).

 * SCons
 * GLFW 3

On Mac, using Homebrew...

```
$ brew install scons glfw3
```

## Run.

From the top level directory...

```
$ scons
```

## Run!

From the top level directory...

```
$ ./runrunrun
```
